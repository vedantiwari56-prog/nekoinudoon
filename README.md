# NekoInuDoon

NekoInuDoon is a simple frontend project for helping stray cats and dogs around Dehradun.
SITE LINK- https://vedantiwari56-prog.github.io/nekoinudoon/

## Meaning of Name

- Neko = Cat
- Inu = Dog
- Doon = Dehradun

## Features

- Responsive homepage
- Intro card with developer contact
- Animal post cards
- Add a new animal post using form
- Search posts by title or location
- Filter posts by animal type
- Delete posts
- Contact button
- Posts saved using localStorage
- Small meow sound when clicking the logo
- Black cat night mode button
- Responsive CSS layout

## Tech Stack

- HTML
- CSS
- JavaScript
- localStorage

## Project Status

This is a frontend prototype. It uses localStorage, so posts are saved in the user's own browser.
A production version would use a real database like Firebase or Supabase.
